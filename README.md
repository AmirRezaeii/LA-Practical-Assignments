# LA Practical Assignments

This repository contains practical implementations of core **Linear Algebra (LA)** concepts. Developed as part of a Linear Algebra course, each Jupyter notebook demonstrates fundamental theories and their applications in data science and machine learning using Python.

The goal is to bridge the gap between mathematical theory and practical coding, providing clear, runnable examples of concepts like matrix decompositions, vector spaces, and dimensionality reduction.

## Repository Structure & Detailed Explanations

| Notebook | Topics Covered |
|----------|----------------|
| **Introduction_to_Numpy.ipynb** | Getting started with NumPy: creating arrays, array operations, broadcasting, indexing, slicing, and basic linear algebra functions (dot product, matrix multiplication). |
| **System_of_Equations,_Gram-Schmidt_and_Change_of_Basis.ipynb** | Three core topics: (1) Solving systems of linear equations using Gaussian elimination and `numpy.linalg.solve`; (2) The Gram-Schmidt process for orthogonalizing a set of vectors; (3) Performing change of basis transformations between different vector space representations. |
| **Diagonalization,_Eigenvectors_and_Decomposition.ipynb** | Computing eigenvalues and eigenvectors, diagonalizing matrices (when possible), understanding the eigenvalue decomposition, and exploring applications like matrix powers and Markov chains. |
| **Latent_Semantic_Analysis.ipynb** | Applying SVD (Singular Value Decomposition) for Latent Semantic Analysis (LSA) – a natural language processing technique that reveals hidden semantic relationships between terms and documents by reducing dimensionality. |

## Key Concepts Illustrated

| Category | Techniques & Topics |
|----------|---------------------|
| **Foundations** | NumPy basics, array operations, broadcasting |
| **Linear Systems** | Solving equations, Gaussian elimination, matrix inversion |
| **Orthogonality** | Gram-Schmidt process, orthogonal bases, change of basis |
| **Spectral Theory** | Eigenvalues, eigenvectors, diagonalization, matrix decompositions |
| **Dimensionality Reduction** | SVD, Latent Semantic Analysis (LSA), principal components |
| **Applications** | NLP (document-term matrices), data compression, topic modeling |
